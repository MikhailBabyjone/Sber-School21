### Тавсияҳо барои лоиҳа
Чӣ гуна дар «Мактаби 21» таҳсил кардан лозим аст:
- Дар тӯли тамоми курс ту бояд мустақилона маълумот ҷустуҷӯ кунӣ. Аз ҳамаи абзорҳои дастрас барои пайдо кардани маълумот истифода бар, масалан, Google ва GigaChat. Ба манбаъҳо бодиққат бош: тафтиш кун, фикр кун, таҳлил кун, муқоиса кун.
- Таълими мутақобила (P2P, Peer-to-Peer) — ин равандест, ки дар он донишомӯзон дониш ва таҷрибаро мубодила мекунанд ва ҳамзамон дар нақши омӯзгор ва шогирд баромад мекунанд. Ин равиш имкон медиҳад, ки на танҳо аз омӯзгор, балки аз якдигар ҳам омӯзӣ, ки ба дарки амиқтари мавод мусоидат мекунад.
- Аз пурсидани кӯмак шарм надор: атрофи ту ҳамон пирҳо ҳастанд, ки онҳо низ ин роҳро бори аввал мегузаранд. Аз посух додан ба дархостҳои кӯмак натарс. Таҷрибаи ту арзишманд ва муфид аст, онро бо дигар иштирокчиён бо ҷуръат тақсим кун.
- Нақл накун, ва агар аз кӯмак истифода барӣ — ҳамеша то охир бифаҳм, ки чаро, чӣ гуна ва барои чӣ. Дар акси ҳол, омӯзиши ту ҳеҷ маъное нахоҳад дошт.
- Агар ту дар чизе дармонда бошӣ ва чунин намояд, ки ҳама чизро санҷидаӣ, вале ҳанӯз ҳам маълум нест, ки ба куҷо равӣ, — танҳо каме дам гир! Бовар кун, ин маслиҳат ба бисёр таҳиягарон дар кори онҳо кӯмак кардааст. Каме ҳаво бихӯр, саратро аз нав фаъол кун, ва шояд дафъаи дигар ба ту ниҳоят ҳалли лозима бирасад!
- На танҳо натиҷаи омӯзиш, балки худи раванд ҳам муҳим аст. На танҳо масъала ҳал кардан лозим, балки фаҳмидан лозим, ки онро ЧӢ ГУНА ҳал кунӣ.
- Ҳангоми иҷрои лоиҳа вақтро назорат кун. Дар як рӯз ту бояд ҳадди ақал як озмоишро паси сар кунӣ.
- Дар хотир дор, ки ҳар як супориш пас аз анҷоми лоиҳа аз як қатор санҷишҳо мегузарад: санҷиши p2p бо ёрии чек-лист, санҷиш бо маҷмӯи автотестҳо, санҷиши услуби код, санҷиш аз ҷониби таҳлилгари статикӣ, санҷиши кори дуруст бо хотира.

Чӣ гуна бо лоиҳа кор кардан лозим аст:
- Маводҳои видеоии муфидро ту метавонед дар бахши Projects (Media) дар Платформа пайдо кунӣ.
- Пеш аз иҷро лоиҳаро бояд аз GitLab ба анбори ҳамном нусха бардорӣ.
- Ҳамаи файлҳои кодиро бояд дар ҷузвдони src/-и анбори нусхабардоришуда эҷод кунӣ.
- Пас аз нусха бардоштани лоиҳа лозим аст шохаи `develop` эҷод кунӣ ва таҳияро дар он пеш барӣ. Баъд аз ин, ба GitLab ҳамчунин шохаи `develop`-ро пуш кардан лозим аст.

## Рӯйхати 1

>Таърихи Linux дар соли 1991 оғоз мешавад, вақте ки аспиранти барномасози финӣ Линус Торвалдс ба таҳияи ядрои системаи оператсионӣ барои компютери худ шурӯъ кард.
>
>Натиҷаҳои кори худро ӯ дар як сервери дастрас барои ҳама ҷой дод, ва ин ба як рӯйдоди калидӣ дар таърихи Linux табдил ёфт. Аввал даҳҳо, баъд садҳо ва ҳазорҳо таҳиягарон лоиҳаи ӯро дастгирӣ карданд — бо кӯшиши муштарак як системаи оператсионии комил ба вуҷуд омад.
>
>Ба Linux системаи Unix хеле таъсир расонд, ин ҳатто аз номаш ҳам маълум аст. Аввалин версияи расмии Linux 1.0 дар соли 1994 баромад. Аз аввал то имрӯз Linux ҳамчун нармафзори озод бо иҷозатномаи GPL паҳн мешавад. Ин маънои онро дорад, ки коди сарчашмаи системаи оператсиониро ҳар як корбар дида метавонад — ва на танҳо дида, балки онро такмил ҳам дода метавонад. Шарти ягона ин аст, ки коди тағйирёфта ва модификатсияшуда низ бояд барои ҳама дастрас бошад ва тибқи иҷозатномаи GPL паҳн шавад. Ин муҳим аст, зеро ба таҳиягарон имкон медиҳад аз код истифода баранд ва ҳамзамон аз мушкилот бо ҳуқуқи муаллиф натарсанд.

## Рӯйхати 3

>Bourne shell (аксар вақт sh аз номи файли иҷрошаванда) — қабати фармонии барвақтаи UNIX аст, ки аз ҷониби Стивен Борн аз Bell Labs таҳия шуда, дар таркиби нашри 7-уми системаи оператсионии UNIX (1978) бароварда шудааст. Ин қабати фармонӣ стандарти де-факто мебошад ва қариб дар ҳар як дистрибутиви *nix дастрас аст.
>
>Номи «bash» ихтисора аз англ. Bourne-again-shell («қабати-фармонии-Борни-дубора») мебошад ва бозии калимаҳо аст: Bourne-shell — яке аз навъҳои маъмули қабати фармонӣ барои UNIX (sh), ки муаллифаш Стивен Борн аст, дар соли 1987 аз ҷониби Брайан Фокс такмил дода шудааст. Насаби Bourne (Борн) бо калимаи англисии born, ки маънои «таваллудшуда»-ро дорад, ҳамоҳангӣ мекунад, аз ин ҷо: қабати фармонии дубора таваллудшуда.

## Рӯйхати 4

>Git — системаи тақсимшудаи идоракунии версияҳо мебошад. Лоиҳа аз ҷониби Линус Торвалдс барои идоракунии таҳияи ядрои Linux сохта шудааст, версияи аввал 7 апрели соли 2005 нашр шуд. Баъдтар ӯ дастгирии онро ба Ҷунио Ҳамано супорид.
>
>То имрӯз — стандарти де-факто дар байни абзорҳои кори дастаҷамъонаи барномасозон аст.
>
>Барнома озод аст ва таҳти иҷозатномаи GNU GPL версияи 2 нашр шудааст. Ба таври пешфарз порти TCP 9418 истифода мешавад.
>
>Таҳияи ядрои Linux дар системаи моликиятии BitKeeper пеш мерафт, ки муаллифаш — Ларри Маквой, худ таҳиягари Linux, — онро ба лоиҳа бо иҷозатномаи ройгон пешниҳод карда буд. Таҳиягарон, барномасозони сатҳи баланд, чанд утилита навиштанд, ва барои яке аз онҳо Эндрю Триджелл реверс-инжиниринги формати интиқоли додаҳои BitKeeper-ро анҷом дод. Дар ҷавоб Маквой таҳиягаронро дар вайрон кардани созишнома айбдор кард ва иҷозатномаро бозпас гирифт, ва Торвалдс ба системаи нав рӯ овард: ҳеҷ яке аз системаҳои кушода ба ҳазорон барномасоз имкон намедод, ки кӯшишҳои худро ҳамоҳанг кунанд (ҳамон низоъ ба навиштани Mercurial овард). Идеология содда буд: равиши CVS-ро гирифта, онро саропо баръакс кардан ва ҳамзамон эътимоднокиро илова намудан.
>
>Таҳияи аввалия камтар аз як ҳафта давом кард: 3 апрели соли 2005 таҳия оғоз шуд, ва аллакай 7 апрел коди Git аз ҷониби системаи нопухта идора мешуд. 16 июн Linux ба Git гузаронида шуд, ва 25 июл Торвалдс аз вазифаҳои таҳиягари пешбари Git даст кашид.
>
>Торвалдс дар бораи номи интихобкардааш git чунин бо лаҳни истеҳзо изҳори назар кард (ки дар сленги англисӣ маънои «бадҷаҳл» дорад): “I'm an egotistical bastard, so I name all my projects after myself. First Linux, now git.”

\> *git clone*

Ту бояд репозиторийро барои клонкунӣ нишон диҳӣ.

    usage: git clone [<options>] [--] <repo> [<dir>]
        -v, --verbose              бештар муфассал бош
        -q, --quiet                камгап бош
        --progress                 гузориши пешрафтро маҷбурӣ намоиш деҳ
        -n, --no-checkout          checkout эҷод накун
        --bare                     репозиторийи холӣ эҷод кун
        --mirror                   репозиторийи оинавӣ эҷод кун (bare-ро дар бар мегирад)
        -l, --local                барои клон кардан аз репозиторийи маҳаллӣ
        --no-hardlinks             аз hardlink-ҳои маҳаллӣ истифода накун, ҳамеша нусха бардор
        -s, --shared               ҳамчун репозиторийи муштарак танзим кун
        --recursive                submodule-ҳоро дар клон ибтидоӣ соз
        --recurse-submodules       submodule-ҳоро дар клон ибтидоӣ соз
        --template <template-directory>     директорияе, ки аз он қолабҳо истифода мешаванд
        --reference <repo>         репозиторийи истинодӣ
        --dissociate               танҳо ҳангоми клонкунӣ аз --reference истифода кун
        -o, --origin <name>        ба ҷои 'origin' <name>-ро барои пайгирии upstream истифода кун
        -b, --branch <branch>      ба ҷои HEAD-и дурдаст <branch>-ро checkout кун
        -u, --upload-pack <path>   роҳ ба git-upload-pack дар сервери дурдаст
        --depth <depth>            клонкунии сатҳӣ бо ҳамин умқ эҷод кун
        --single-branch            танҳо як шоха, HEAD ё --branch-ро клон кун
        --separate-git-dir <gitdir>       git dir-ро аз дарахти корӣ ҷудо кун
