Таърихи забони C

Забони барномасозии C дар аввали солҳои 1970 аз ҷониби Деннис М. Ритчи, корманди Bell Labs (AT&T), таҳия шуд.

Дар солҳои 1960 Ритчи ҳамроҳ бо чанд корманди дигари Bell Labs (AT&T) рӯйи лоиҳае бо номи Multics кор мекард. Ҳадафи лоиҳа таҳияи системаи оператсионӣ барои як компютери бузург буд, ки онро ҳазор корбар истифода бурда метавонистанд. Соли 1969 AT&T (Bell Labs) аз лоиҳа хориҷ шуд, зеро лоиҳа наметавонист як системаи аз ҷиҳати иқтисодӣ судманд тавлид кунад. Аз ин рӯ, кормандони Bell Labs (AT&T) бояд лоиҳаи дигари корӣ меҷустанд (асосан Деннис М. Ритчи ва Кен Томпсон).

Кен Томпсон ба таҳияи системаи нави файл оғоз кард. Ӯ як версияи системаи нави файлро барои DEC PDP-7 бо ассемблер навишт. (Системаи нави файл ҳамчунин барои бозии Space Travel истифода мешуд). Дере нагузашта онҳо ба беҳтарсозӣ ва илова кардани васеъшавиҳо шурӯъ карданд. (Онҳо дониши худро аз лоиҳаи Multics барои илова кардани беҳбудиҳо истифода бурданд). Баъд аз чанде як системаи комил ба вуҷуд омад. Брайан В. Керниган ин системаро UNIX номид, ки ишораи ҳаҷвомез ба Multics буд. Тамоми система ҳанӯз ҳам бо коди ассемблерӣ навишта шуда буд.

Илова бар ассемблер ва Fortran, UNIX инчунин як тафсиргар барои забони барномасозии B дошт. (Забони B бевосита аз BCPL-и Мартин Ричардс гирифта шудааст). Забони B-ро соли 1969–70 Кен Томпсон таҳия кард. Дар рӯзҳои аввал коди компютерӣ бо коди ассемблерӣ навишта мешуд. Барои иҷрои як вазифаи мушаххас, бояд саҳифаҳои зиёди код навишт. Забони сатҳи баланд мисли B имкон медод, ки ҳамон вазифа танҳо бо чанд сатри код навишта шавад. Забони B барои рушди минбаъдаи системаи UNIX истифода мешуд. Бо сабаби сатҳи баланди забони B, кодро хеле зудтар аз ассемблер тавлид кардан мумкин буд.

Як нуқсони забони B ин буд, ки он навъҳои додаҳоро намешинохт. (Ҳама чиз бо калимаҳои мошинӣ ифода мешуд). Функсионалияти дигари забони B, ки дастрас набуд, истифодаи «структураҳо» буд. Норасоии ҳамин чизҳо сабаб шуд, ки Деннис М. Ритчи забони барномасозии C-ро таҳия кунад. Ҳамин тавр, дар солҳои 1971–73 Деннис М. Ритчи забони B-ро ба забони C табдил дод ва бештар синтаксиси забони B-ро нигоҳ дошт, дар ҳоле ки навъҳои додаҳо ва бисёр тағйироти дигарро илова кард. Забони C омезиши пурқудрати функсионалияти сатҳи баланд ва хусусиятҳои дақиқеро дошт, ки барои барномасозии системаи оператсионӣ зарур буданд. Аз ҳамин сабаб, бисёре аз ҷузъҳои UNIX дар ниҳоят бо C аз нав навишта шуданд (худи ядрои Unix соли 1973 дар DEC PDP-11 аз нав навишта шуд).

Забони барномасозии C-ро Керниган ва Ритчи дар китоби ҳоло классикӣ бо номи «The C Programming Language, 1st edition» сабт карданд. (Керниган гуфтааст, ки ӯ дар тарҳрезии забони C нақше надоштааст: «It’s entirely Dennis Ritchie’s work». Аммо ӯ муаллифи барномаи машҳури «Hello, World» ва бисёр барномаҳои дигари UNIX аст).

Солҳо китоби «The C Programming Language, 1st edition» стандарти забони C буд. Соли 1983 Кумитаи Институти Миллии Стандартҳои Амрико (ANSI) барои таҳияи таърифи муосир барои забони барномасозии C таъсис дода шуд (ANSI X3J11). Соли 1988 онҳо таърифи ниҳоии стандартии ANSI C-ро пешниҳод карданд. (Ин стандарт бар асоси китоби K&R, нашри 1 сохта шуда буд).

Стандарти ANSI C ба тарҳи аслии забони C тағйироти хеле кам ворид кард. (Онҳо бояд боварӣ ҳосил мекарданд, ки барномаҳои кӯҳна бо стандарти нав ҳамчунон кор мекунанд). Баъдтар стандарти ANSI C аз ҷониби Созмони Байналмилалии Стандартсозӣ (ISO) қабул шуд. Аз ин рӯ, истилоҳи дуруст бояд ISO C бошад, аммо ҳама ҳанӯз ҳам онро ANSI C меноманд.
