# Дастур барои санҷиш

Илова бар он ки барнома маълумоти дурустро бармегардонад, автотест коди барномае, ки ту навиштаӣ, аз рӯйи нуқтаҳои зерин месанҷад:

* **Услуби код.** Барои санҷидани он ки коди зебое, ки ту навиштаӣ, то чӣ андоза ба стандарт мувофиқ аст, ту метавонӣ аз утилитаи _clang-format_ истифода барӣ.
 

```materials/linters``` папкада ```.clang-format``` файли мавҷуд аст, ки дар он танзимоти зарурии услуб нигоҳ дошта шудаанд.
Ин файл ба ҳамаи файлҳое, ки дар ҳамин папка ва зерпапкаҳои он ҷойгиранд, таъсир мерасонад.
Барои истифодаи ин танзимот дар файлҳои худ, ин файлро ба папкаи ```
src

``` нусхабардорӣ кун. \
 \
 Барои санҷидани услуб аз фармони зерин истифода бар: \
 ```
clang-format -n src/sourcefile_name.c```

 Версияи зарурии clang-format: \
 **Linux** 18.1.8
