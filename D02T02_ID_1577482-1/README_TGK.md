## Муқаддима
### Тавсияҳо барои лоиҳа

Чӣ тавр дар «Мактаби 21» таҳсил кардан:
- Дар тӯли тамоми курс ту мустақилона маълумот меҷӯӣ. Аз ҳама воситаҳои дастрас барои ҷустуҷӯи маълумот истифода бар, масалан, Google ва GigaChat. Ба манбаъҳои маълумот бодиққат бош: тафтиш кун, фикр кун, таҳлил кун, муқоиса кун.
- Таълими мутақобила (P2P, Peer-to-Peer) — ин равандест, ки дар он донишомӯзон дониш ва таҷрибаро мубодила мекунанд ва ҳамзамон дар нақши омӯзгор ва шогирд баромад мекунанд. Ин равиш имкон медиҳад, ки на танҳо аз омӯзгор, балки аз якдигар ҳам омӯзӣ, ки ба дарки амиқтари мавод мусоидат мекунад.
- Аз пурсидани кӯмак шарм надор: атрофи ту ҳамон гуна пирҳо ҳастанд, ки онҳо ҳам ин роҳро бори аввал тай мекунанд. Аз посух додан ба дархостҳои кӯмак натарс. Таҷрибаи ту арзишманд ва фоиданок аст, онро бо дигар иштирокчиён далерона тақсим кун.
- Нақл накун, ва агар аз кӯмак истифода барӣ — ҳамеша то охир фаҳм, ки чаро, чӣ тавр ва барои чӣ. Дар акси ҳол, омӯзиши ту ҳеҷ маъное нахоҳад дошт.
- Агар дар чизе дармонда бошӣ ва ба назар расад, ки ҳама чизро санҷидаӣ, вале ҳанӯз ҳам намефаҳмӣ, ки ба куҷо равӣ, — фақат каме дам гир! Бовар кун, ин маслиҳат ба бисёр таҳиягарон дар кори онҳо кӯмак кардааст. Каме ҳаво бихӯр, саратро аз нав фаъол кун, ва шояд дафъаи дигар ниҳоят ҳалли лозимӣ ба ту бирасад!
- На танҳо натиҷаи омӯзиш муҳим аст, балки худи раванд ҳам. На танҳо масъалаеро ҳал кардан лозим, балки фаҳмидан лозим, КӢ ТАВР онро ҳал кунӣ.
- Ҳангоми иҷрои лоиҳа вақтро назорат кун. Дар як рӯз ту бояд ҳадди ақал як озмоишро паси сар кунӣ.
- Дар хотир дор, ки ҳар як супориш пас аз анҷоми лоиҳа як қатор санҷишҳоро мегузарад: санҷиши p2p бо ёрии рӯйхати тафтиш, санҷиш бо маҷмӯи автотестҳо, санҷиши услуби код, санҷиш бо таҳлилгари статикӣ, санҷиши кори дуруст бо хотира.

Чӣ тавр бо лоиҳа кор кардан:
- Маводҳои видеоии муфидро ту метавонед дар бахши Projects (Media) дар Платформа пайдо кунӣ.
- Пеш аз иҷро лоиҳаро бояд аз GitLab ба репозиторияи ҳамном клон кунӣ.
- Ҳамаи файлҳои кодиро бояд дар папкаи src/-и репозиторияи клоншуда эҷод кунӣ.

## Рӯйхат 1

>Забони барномасозии C (англ. C) соли 1973 аз ҷониби Деннис Ритчи дар AT&T Bell Laboratories ҳамчун забони барномасозии сатҳи баланд барои таҳияи системаи амалиётии UNIX офарида шуд. Баъдтар он ба бисёр платформаҳои дигар ҳам интиқол дода шуд.
>
>Забони C — забони барномасозии мурофиавӣ (сохторӣ) аст. Барнома дар C аз маҷмӯи мурофиаҳо ё функсияҳо иборат аст. Ин мурофиаҳо ва функсияҳо дастурҳои гуногунро дар бар мегиранд: эълонҳо, операторҳо ва унсурҳои дигаре, ки дар маҷмӯъ алгоритми кори системаи ҳисоббарорро тавсиф мекунанд. Номи нуқтаи вуруд ба барнома — main (функсияи асосӣ) аст. Он метавонад ҳам параметрҳои расмӣ надошта бошад ва ҳам аз сатри фармон маълумот қабул кунад. Тани функсияҳо дар шакли дастури таркибӣ тавсиф мешавад ва дар қавсҳои фигурӣ ( { } ) ҷойгир мешавад. C бо тағйирёбандаҳо кор мекунад, ки бояд пеш аз истифодаашон дар коди барнома эълон шаванд. Ҳамаи дастурҳои содаи забони C бо нуқта ва вергул анҷом меёбанд. Ҳар як ифода бо нуқта ва вергул дастур эҷод мекунад.
>
>Шарҳҳо бо аломатҳои « /* » оғоз мешаванд ва бо аломатҳои « */ » анҷом меёбанд ва метавонанд чанд сатрро ишғол кунанд. Ҳамчунин шарҳҳои яксатрӣ вуҷуд доранд, ки бо аломатҳои « // » оғоз мешаванд. Шарҳҳо ягон кори фоиданок иҷро намекунанд — онҳо барои он лозиманд, ки ба таҳиягарон фаҳмидани код осонтар шавад. Бо онҳо аз ҳад зиёд машғул шудан лозим нест, беҳтарин код — ин кодест, ки худ аз худ гап мезанад, бе шарҳҳо...

Ба `git` танҳо файлҳои сарчашма бо код ва файлҳои додаҳо (агар лозим бошад) ба директорияи `src`, шохаи `develop`, push кардан лозим аст. Дар ёддошт гуфта шуда буд, ки ӯро асабонӣ накунӣ, то дар кушодани дар ба ту кӯмак кунад.

Аммо, ба назар мерасад, ин бача худ мушкилро аз ҳеҷ ҷо меҷӯяд.

Дар ҳар сурат, бояд модули намоишро ислоҳ кунӣ ва ба он имкони ворид кардани номро аз худи аввал, тавассути `stdin`, илова намоӣ. Ва барориш дар ин ҳолат бояд ба чунин шакл тағйир ёбад: «Hello, <name>!». Ва фаромӯш накун, ки ИИ — барнома аст, пас номаш ҳам бояд рақамӣ бошад.
